[SIZE=5][B]🔥 Why RedstoneReboot?[/B][/SIZE]

RedstoneReboot isn't just a restart plugin — it's a [B]professional-grade server lifecycle engine[/B] built for production environments. From single-server setups to massive networks, it delivers reliability, intelligence, and elegance.

This resource page provides builds for Bukkit-family servers and Folia. [I](Fabric, Forge, and NeoForge mod variants are distributed separately directly from Modrinth and GitHub)[/I]

[HR][/HR]

[SIZE=5][B]✨ Core Capabilities[/B][/SIZE]

[LIST]
[*] [B]Intelligent Scheduling[/B] — Multiple daily restart windows per day, complete with interval warnings.
[*] [B]Global Timezones[/B] — Configure timezones and day-of-week filters (e.g. Asia/Kolkata, UTC).
[*] [B]Actionable Performance Tracking[/B] — Live TPS and memory monitoring with strict threshold triggers.
[*] [B]Emergency Fail-safes[/B] — Graceful server-saving stops when critical memory leaks or TPS hits occur.
[*] [B]Rich Multi-channel Alerts[/B] — Chat, beautiful Titles, Action Bars, and Sound indicators.
[*] [B]Complete Folia Support[/B] — Dedicated build explicitly designed around region-threaded logic.
[/LIST]

[HR][/HR]

[SIZE=5][B]⚙️ Installation[/B][/SIZE]

[LIST=1]
[*] Download the correct plugin JAR (Bukkit for standard, Folia for Region-Threaded).
[*] Place it into your server's [CODE]plugins/[/CODE] folder.
[*] Start the server.
[*] Edit [CODE]plugins/RedstoneReboot/config.yml[/CODE].
[*] Run [CODE]/reboot reload[/CODE] or restart server to apply.
[/LIST]

[HR][/HR]

[SIZE=5][B]🎮 Commands & Permissions[/B][/SIZE]

[COLOR=#DC2626]/reboot[/COLOR] — Plugin status & help
[COLOR=#DC2626]/reboot now [delay][/COLOR] — Restart immediately
[COLOR=#DC2626]/reboot schedule <seconds>[/COLOR] — Schedule future restart
[COLOR=#DC2626]/reboot cancel[/COLOR] — Cancel pending countdown
[COLOR=#DC2626]/reboot status[/COLOR] — Show schedule status
[COLOR=#DC2626]/reboot info[/COLOR] — Show health diagnostics
[COLOR=#DC2626]/reboot reload[/COLOR] — Hot-reload configs

[B]Integrations[/B]: Full support for [B]PlaceholderAPI[/B] (next restart, time until, status) and [B]LuckPerms[/B] permissions.
