[SIZE=5][B]🔥 What is RedstoneReboot?[/B][/SIZE]

RedstoneReboot is the most advanced, production-grade restart management system for Minecraft servers. It supports [B]every major platform[/B] and Minecraft versions [B]1.9 through 1.21.1[/B].

Whether you run a small SMP or a massive network, RedstoneReboot provides the reliability, intelligence, and elegance your server deserves.

[HR][/HR]

[SIZE=5][B]✨ Features[/B][/SIZE]

[LIST]
[*] [B]🕐 Smart Scheduling[/B] — Multiple daily restart windows with timezone & day-of-week support
[*] [B]📊 TPS & Memory Monitoring[/B] — Automatic restart triggers when performance degrades
[*] [B]🚨 Emergency Restarts[/B] — Instant action on critical server conditions (TPS < 12, Memory > 95%)
[*] [B]🔔 Multi-Channel Alerts[/B] — Chat, Title, ActionBar, Sound — all fully customizable
[*] [B]🔗 PlaceholderAPI[/B] — 10+ placeholders for scoreboards, MOTDs, tab lists
[*] [B]🔐 LuckPerms[/B] — Deep permission integration (no hard dependency)
[*] [B]🛠️ Developer API[/B] — Events, hooks, programmatic restart control
[*] [B]🧩 Multi-Platform[/B] — Bukkit, Spigot, Paper, Folia, Fabric, Forge, NeoForge
[/LIST]

[HR][/HR]

[SIZE=5][B]🎮 Commands[/B][/SIZE]

[CODE]
/reboot              — Show plugin status
/reboot now [delay]  — Restart with countdown
/reboot schedule <s> — Schedule future restart
/reboot cancel       — Cancel pending restart
/reboot status       — Detailed status view
/reboot info         — Performance information
/reboot reload       — Hot-reload config
[/CODE]

[HR][/HR]

[SIZE=5][B]📊 PlaceholderAPI[/B][/SIZE]

[CODE]
%redstonereboot_next_restart%  — Next restart time
%redstonereboot_time_until%    — Time remaining
%redstonereboot_tps%           — Current TPS
%redstonereboot_memory%        — Memory usage
%redstonereboot_status%        — Server status
[/CODE]
