## ⚡ RedstoneReboot — Advanced Multi-Platform Server Restart Engine

The most advanced, production-grade restart management system ever built for Minecraft servers. Trusted by networks running hundreds of players.

---

### 🔥 Core Features

**🕐 Intelligent Scheduling**
Schedule multiple daily restarts across any timezone. Supports day-of-week filtering, configurable warning countdowns, and seamless timezone conversion.

**📊 Real-Time TPS & Memory Monitoring**
Continuously tracks your server's TPS and memory usage. Automatically triggers restarts when performance degrades, with configurable thresholds and consecutive-check requirements to avoid false positives.

**🚨 Emergency Restart System**
When your TPS drops below critical levels or memory spikes dangerously, RedstoneReboot immediately warns players and executes a graceful emergency shutdown — saving worlds and player data before going down.

**🔔 Multi-Channel Alerts**
Warn players through **Chat Messages**, **Screen Titles**, **Action Bars**, and **Sounds** — all independently configurable with full color code support.

**🧩 Multi-Platform**
Runs on Paper, Folia, Bukkit, Spigot — with versions spanning **1.9 through 1.21.1**. Fabric, Forge, and NeoForge mod variants also available.

**🔗 Integrations**
- **PlaceholderAPI**: 10+ placeholders for scoreboards, MOTDs, tab lists
- **LuckPerms**: Deep permission integration via reflection (no hard dependency)
- **Developer API**: Full API with events, hooks, and programmatic control

---

### 📋 Commands

| Command | Description |
|---------|-------------|
| `/reboot` | Show status & help |
| `/reboot now [delay]` | Restart with optional countdown |
| `/reboot schedule <seconds>` | Schedule a future restart |
| `/reboot cancel` | Cancel pending restart |
| `/reboot status` | Detailed restart status |
| `/reboot reload` | Hot-reload configuration |

### 📦 Requirements

- **Paper / Spigot**: 1.9 — 1.21.1
- **Folia**: 1.20+
- **Java**: 17+ (8+ for legacy MC versions)
