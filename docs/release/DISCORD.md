# Discord Release Message

**⚡ RedstoneReboot `<version>` is out.**

This release adds bStats metrics integration, fixes PlaceholderAPI MOTD compatibility, and ships a complete documentation overhaul across all marketplace pages.

**Highlights:**
- 📈 **bStats Integration** — anonymous server metrics at <https://bstats.org/plugin/bukkit/RedstoneReboot/30751>
- 🧩 **PlaceholderAPI MOTD Fix** — placeholders now work reliably in server-list MOTD plugins
- 🩺 **Doctor Command** — `/reboot doctor` for backend & environment diagnostics
- 🔄 **Hot-Reload** — change backend config and `/reboot reload` without a full server restart
- 🕐 Scheduled restarts with global timezone support
- 📊 TPS and memory-based restart automation
- 🚑 Emergency restart handling & memory leak prevention
- 📦 Builds for Bukkit, Paper, Folia, Fabric, Forge, and NeoForge
