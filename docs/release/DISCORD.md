# Discord Release Message

**⚡ RedstoneReboot `<version>` is out.**

This release tightens the multi-platform stack across Bukkit, Folia, Fabric, Forge, and NeoForge. 
We've introduced verified builds, cleaner scheduler behavior on non-Bukkit loaders, and refreshed platform-facing docs for distribution pages.

**Highlights:**
- 🕐 Scheduled restarts with global timezone support
- 📊 TPS and memory-based restart automation
- 🚑 Emergency restart handling & memory leak prevention
- 📦 Plugin builds for Bukkit-family servers and Folia
- 🧩 Native Mod builds for Fabric, Forge, and NeoForge
